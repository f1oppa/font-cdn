## Segoe UI font CDN
To use, add this to your code:
```html
<link rel="stylesheet" href="https://f1oppa.github.io/cdn/fonts.css">
```
You can choose from these styles:
- Segoe UI Regular
- Segoe UI Italic
- Segoe UI Bold
- Segoe UI Bold Italic
## Example
```html
<html>
  <head>
    <link rel="stylesheet" href="https://f1oppa.github.io/cdn/fonts.css">
    <style>
      body {
        font-family: "Segoe UI Regular";
      }
    </style>
  </head>
  <body>
    Hello, World!
  </body>
</html>
```
